Politique de Confidentialité

Informations sur la société

Le site web Lëc Média est détenu par Lëc Média, qui est un contrôleur de données de vos données personnelles.

Nous avons adopté cette politique de confidentialité, qui détermine la manière dont nous traitons les informations collectées par Lëc Média, qui fournit également les raisons pour lesquelles nous devons collecter certaines données personnelles vous concernant.

Par conséquent, vous devez lire cette politique de confidentialité avant d'utiliser le site web de Lëc Média.

Collecte de données personnelles

Nous prenons soin de vos données personnelles et nous nous engageons à en garantir la confidentialité et la sécurité.

Les informations personnelles que nous collectons comprennent :

  • Informations sur votre appareil (navigateur web, adresse IP, etc.)
  • Informations sur votre interaction avec le site (pages visitées, produits consultés, etc.)
  • Données que vous nous fournissez lors de l'inscription (nom, prénom, adresse, etc.)

Finalités du traitement des données

Nous traitons vos données uniquement dans la mesure où cela est absolument nécessaire pour maintenir le site web et pour des raisons légitimes telles que la sécurité et l'analyse statistique.

Vos droits

Si vous êtes un résident européen, vous disposez de certains droits concernant vos données personnelles, notamment :

  • Le droit d'être informé
  • Le droit d'accès
  • Le droit de rectification
  • Le droit à l'effacement
  • Le droit de restreindre le traitement
  • Le droit à la portabilité des données
  • Le droit d'opposition

Pour exercer ces droits ou pour toute question relative à vos données personnelles, veuillez nous contacter à l'adresse e-mail : info@lec-media.agency.

Liens vers d'autres sites web

Notre site web peut contenir des liens vers d'autres sites web dont nous ne sommes pas responsables des pratiques de confidentialité. Nous vous encourageons à lire les politiques de confidentialité de ces sites tiers.

Sécurité de l'information

Nous sécurisons les informations que vous nous fournissez et nous mettons en place des mesures pour protéger vos données contre tout accès non autorisé.

Divulgation légale

Nous divulguerons vos informations si cela est requis par la loi ou si nous croyons de bonne foi que la divulgation est nécessaire pour protéger nos droits ou répondre à une demande gouvernementale.

Contactez-nous pour toute question relative à notre politique de confidentialité : info@lec-media.agency
